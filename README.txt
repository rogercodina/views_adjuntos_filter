Views adjuntos filter
=====================

Creates a custom exposed filter for views: This exposed form allows users to
filter content with attached documents or without them. The field which is
considered for this logic is "field_documentos_adjuntos".

In this module you can find an example of a subquery. Read more about
subquering on Drupal:

http://drupal.stackexchange.com/questions/18497/how-do-i-use-not-in-in-a-query
https://api.drupal.org/comment/56228#comment-56228
